# browerai-js-parser

JavaScript Parser for BrowerAI

## Overview

Parse JavaScript code into an AST using the Boa parser. Pure Rust implementation, no V8 dependency.

## Features

- ECMAScript standard support
- Pure Rust implementation (via Boa)
- AST generation
- JavaScript execution via Boa engine

## Usage

```rust
use browerai_js_parser::*;

// Example usage
```

## Tests

Run tests with:

```bash
cargo test -p browerai-js-parser
```
