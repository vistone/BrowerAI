# browerai-ai-integration

AI Integration for BrowerAI

## Overview

Integration layer connecting AI models with parsers. Provides AI-enhanced parsing capabilities.

## Features

- HTML model integration for structure validation
- CSS model integration for style optimization
- JavaScript model integration for deobfuscation
- Graceful fallback when models unavailable

## Usage

```rust
use browerai_ai_integration::*;

// Example usage
```

## Tests

Run tests with:

```bash
cargo test -p browerai-ai-integration
```
