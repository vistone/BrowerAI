# browerai-css-parser

CSS Parser for BrowerAI

## Overview

Parse CSS stylesheets and selectors. Handles CSS tokenization, selector parsing, and property value parsing.

## Features

- CSS3 selector support
- Property value parsing
- Selector compilation
- Style computation

## Usage

```rust
use browerai_css_parser::*;

// Example usage
```

## Tests

Run tests with:

```bash
cargo test -p browerai-css-parser
```
