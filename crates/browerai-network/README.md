# browerai-network

Network Layer for BrowerAI

## Overview

HTTP client, caching, and web crawling capabilities.

## Features

- HTTP/HTTPS client with reqwest
- Resource caching
- Deep web crawling
- URL parsing and normalization

## Usage

```rust
use browerai_network::*;

// Example usage
```

## Tests

Run tests with:

```bash
cargo test -p browerai-network
```
