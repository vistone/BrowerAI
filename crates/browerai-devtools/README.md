# browerai-devtools

Development Tools for BrowerAI

## Overview

Developer tools for debugging and profiling.

## Features

- DOM inspector
- Network monitor
- Performance profiler
- Request/response tracking

## Usage

```rust
use browerai_devtools::*;

// Example usage
```

## Tests

Run tests with:

```bash
cargo test -p browerai-devtools
```
