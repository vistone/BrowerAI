# browerai-plugins

Plugin System for BrowerAI

## Overview

Extensible plugin architecture for adding custom functionality.

## Features

- Plugin trait and registry
- Dynamic plugin loading
- Plugin lifecycle management
- Hook system for extensibility

## Usage

```rust
use browerai_plugins::*;

// Example usage
```

## Tests

Run tests with:

```bash
cargo test -p browerai-plugins
```
