# browerai-testing

Testing Utilities for BrowerAI

## Overview

Comprehensive testing framework for benchmarking and website testing.

## Features

- Benchmark runner with timing and metrics
- Website testing suite
- Comparison tools for before/after analysis
- Performance regression detection

## Usage

```rust
use browerai_testing::*;

// Example usage
```

## Tests

Run tests with:

```bash
cargo test -p browerai-testing
```
