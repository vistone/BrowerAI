# browerai-renderer-core

Renderer Core for BrowerAI

## Overview

Core rendering engine including layout calculation and paint operations.

## Features

- Box model computation
- Flexbox and Grid layout support
- Paint operations (shapes, colors, opacity)
- Style application and cascade resolution

## Usage

```rust
use browerai_renderer_core::*;

// Example usage
```

## Tests

Run tests with:

```bash

```
