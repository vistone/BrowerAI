# browerai-html-parser

HTML5 Parser for BrowerAI

## Overview

Standards-compliant HTML5 parsing using html5ever. Produces a DOM tree that can be traversed and manipulated.

## Features

- HTML5 specification compliant
- Handles malformed HTML gracefully
- Streaming parser for memory efficiency
- Optional AI enhancement for structure prediction

## Usage

```rust
use browerai_html_parser::*;

// Example usage
```
